# TontiPay
Cotisez. Ramassez. En confiance.

> **"Une commerçante qui vérifie son téléphone entre deux clients doit comprendre en 3 secondes."**

TontinePro est une application mobile-first conçue pour simplifier la gestion des tontines (épargne rotative). Elle remplace la complexité technique par une interface intuitive basée sur la confiance et la transparence.

---

## 🚀 Vision du Projet
- **Simplification Radical :** Navigation limitée à 4 onglets maximum.
- **Zéro Jargon :** Des termes clairs comme "Mon Argent" au lieu de "Flux de trésorerie".
- **Design Prestige :** Une identité visuelle "Nana Benz" (Orange, Brun Nuit, Blanc Chaud).
- **Focus Action :** Un seul objectif principal par écran.

## 🛠️ Stack Technique
- **Framework :** [React 19](https://react.dev/) + [Vite 6](https://vitejs.dev/)
- **Style :** [Tailwind CSS 4](https://tailwindcss.com/) (Couleurs OKLCH)
- **Icônes :** [Lucide React](https://lucide.dev/)
- **Déploiement :** [Vercel](https://vercel.com/)
